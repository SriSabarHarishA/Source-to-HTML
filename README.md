# Source2HTML

## Project Description

Source2HTML is a mini project developed using C language.

The objective of this project is to convert C source code into an HTML file with syntax highlighting.

The generated HTML file can be opened in any web browser to view colorful formatted source code similar to code editors.

---

# Features

- Converts C source code to HTML
- Syntax highlighting support
- Keyword highlighting
- Comment highlighting
- String highlighting
- Numeric constant highlighting
- ASCII character highlighting
- Preprocessor directive highlighting
- CSS based styling
